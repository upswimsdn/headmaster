CMSI 387

Operating Systems

1. Become an operating system "power user."
  • 1a. Effectively and efficiently use a command-based operating system shell.
  • 1b. Monitor and manage processes and memory use.
  • 1c. Navigate, monitor, and manage a logical file system.
  • 1d. Redirect input and output streams to and from files, processes, and networked computers.
  • 1e. Interact with operating systems across the network.
2. Follow academic and technical best practices.
  • 2a. Write syntactically correct, functional code.
  • 2b. Demonstrate proper separation of concerns.
  • 2c. Write code that is easily understood by programmers other than yourself
  • 2d. Use available resources and documentation to find required information.
  • 2e. Use version control effectively.
